# Source Evaluation

## Evidence Ledger

Maintain a compact ledger for every substantive source.

| Field | Meaning |
| --- | --- |
| id | Stable ID such as S1, S2 |
| claim_ids | Claims the source supports or refutes |
| name | Source name |
| url | URL when available |
| accessed | Access date for live/current sources |
| tier | Reliability tier, 1 highest to 8 lowest |
| independence | Independent, same-origin, syndicated, or unknown |
| site_rating | Established, credible-limited, questionable, unreliable, cannot-assess |
| author | Named author or unidentified |
| evidence_quality | Strong, moderate, weak, fabricated/misrepresented |
| contribution | What this source proves, disproves, or contextualizes |
| access_note | Blocked, paywalled, snippet-only, archived, or full access |

## Source Tiers

| Tier | Source Type | Use |
| --- | --- | --- |
| 1 | Established fact-check organizations | Strong for claim history and debunks |
| 2 | Official institutional sources | Strong for law, policy, public health, statistics |
| 3 | Major wire services and quality journalism | Strong for current events when sourced |
| 4 | Peer-reviewed science or systematic reviews | Strong for scientific/medical claims |
| 5 | Domain expert organizations | Useful for technical or sector context |
| 6 | Other media with editorial standards | Use with corroboration |
| 7 | Blogs, social media, personal sites | Supplement only; not enough for high confidence |
| 8 | Anonymous, unattributed, imitation, or opaque sources | Treat as very low reliability |

High confidence normally requires at least three independent Tier 1-4 sources that
directly address the claim. If fewer sources exist, say so.

## Lateral Reading Protocol

Do not judge a source only by how polished it looks. Leave the page and check:

1. What independent sources say about the site.
2. Whether the author is real and has relevant expertise.
3. Whether citations are primary and accurately represented.
4. Whether the claim appears in reliable sources or only in low-tier networks.
5. Whether the domain imitates a known outlet or hides ownership.

Useful searches:

- `"site name" reliability`
- `"site name" fact check`
- `"author name" credentials`
- exact claim in quotes
- neutral claim formulation
- opposite claim formulation

## CRAAP Checks

- Currency: Is it current enough for this topic?
- Relevance: Does it directly address the claim?
- Authority: Who produced it and with what expertise?
- Accuracy: Are methods, data, and citations inspectable?
- Purpose: Inform, persuade, sell, entertain, or mobilize?

## Access Problems

When a source is blocked, paywalled, CAPTCHA-protected, or unavailable:

- Do not silently drop it.
- Search for the headline, archived copies, syndicated versions, and fact-check
  summaries.
- Mark the source as snippet-only or blocked in the ledger.
- Lower confidence if it is a key source and no substitute exists.

## Independence

Three articles repeating the same wire copy or quoting the same anonymous post are
not three independent confirmations. Count independence by evidence origin, not by
the number of URLs.
